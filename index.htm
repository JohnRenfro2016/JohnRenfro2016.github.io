Memorizing Kanji

  • Use 10ten Japanese Reader browser extension to read kanji on your desktop or laptop computer.
  • Group kanji by the same sound using WaniKani.
  • Group them by similar elements.
  • Practice writing only the same sound Kanji until you can do it from memory using Android Akebi.
